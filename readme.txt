Подключение БД
1. В SQL Server Management Studio подключитесь к экземпляру SQL Server Компонент Database Engine.
2. Щелкните по "базу данных" правой кнопкой мыши.
3. Выберите присоединить.
4. Выберите файл базы данных с расширением .mdf.
5. Нажмите ок.

Подключение проекта
1. Устанавливаем среду разработки PyCharm
2. Щелкните File -> Open
3. Выберите нужный проект
4. Перенесите файл модели с расширениями .h5 .json в проект
5. Установите все необходимые библиотеки.
